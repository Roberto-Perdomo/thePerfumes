#  Perfume Store App

Aplicación web para explorar y visualizar perfumes. Permite ver productos, detalles e imágenes a través de una interfaz moderna conectada a una API.

---

## 🚀 Tecnologías utilizadas

### Frontend
- Vue.js
- Vue Router
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express
